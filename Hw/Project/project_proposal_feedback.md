Overall I think this is a good project proposal, but possibly too much for the time limit. What I might suggest is, if you are not 100% rock solid on the NoahMP translation to Pytorch, then focus on developing the routing model. You can use the gridded met data to run a simpler ML model (e.g. LSTM) to predict the runoff at grid cells and then use that as the input to the GNN routing model. It should still be transferrable to your research, but you might not have to rely on the NoahMP translation being perfect, and will likely be more computationally efficient. If you have time, you can then add the NoahMP translation as a second part of the project, but I would focus on the routing model first.
